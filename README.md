# Annotation Sample

## Task:
Identify whether the sentence is clear or unclear.

---

*Sentence:*  
The report was kind of confusing in some parts.

*Label:*  
Unclear

*Reason:*  
The phrase "kind of confusing" is vague and does not specify which parts are problematic.

---

*Sentence:*  
The report lacks clarity in the financial analysis section due to missing data.

*Label:*  
Clear

*Reason:*  
The issue is specific and clearly explained.
